jQuery UI: Draggable

Instructions:

Part 1:

  1. Create 2 draggable objects using the images, "images/earth.jpg" and "images/tulips.jpg".
  2. Constrain one to move only horizontally and the other move only vertically.
  3. Specify the "crosshair" cursor for one, and the "pointer" for the other.

Part 2:

  1. Create a draggable object using the image "images/tiger.jpg".
  2. Constrain it so that it can only be moved inside its container (#draggableContainer).
