jQuery UI: Custom Widget

Instructions:

Part 1:

  1. Create a custom widget called "counter".
  2. It should have 2 options: "value" and "increment_amount".
    • "value" should be a number that will be incremented and decremented.
    • "increment_amount" is the amount that a number will be incremented
  3. When it is created, it should fill its element's text with "value".
  4. It should have 2 methods: "increment" and "decrement".
    • "increment" should increase the "value" by the "increment_amount"
    • "decrement" should decrease the "value" by the "increment_amount"
    • Both methods should update the element's text with the new value.
  5. Make the "-" button call the "decrement" method on the widget.
  6. Make the "+" button call the "increment" method on the widget.
